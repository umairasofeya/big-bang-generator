# Big Bang Generator

## Requirements
Have Node.js installed

## Instructions on how to run from Terminal:

1. Open the Terminal application on your device.

2. Clone the repository:
```bash
git clone <github-repo-link> umaira-bigbang
```
3. Enter the project folder to access its contents.
```bash
cd umaira-bigbang
```

4. Run the script using this command below. 
```bash
node bigbang.js
```

5. The file output.json is generated in the project folder.
